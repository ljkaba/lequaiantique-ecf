# Le quai antique

Application Web vitrine pour un restaurant.

## Installation

* Cloner le dépôt
* Installer PHP 8.1
* Installer Composer
* Installer Symfony CLI
* Installer Docker
* Installer Docker Compose

## Lancement

* Lancer le serveur de développement avec `symfony serve -d`
* Lancer les conteneurs Docker avec `docker-compose up -d`
